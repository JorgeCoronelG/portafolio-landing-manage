## Uso

Instalar las dependencias (Tailwind)

```
npm install
```

Ejecutar el compilador de Tailwind CLI de **input.css** durante el desarrollo. El archivo css de salida será **css/main.css**
```
npm watch
```

Para construir una sola vez el archivo de estilos
```
npm build
```
